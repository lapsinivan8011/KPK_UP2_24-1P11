# Вариант №12. Curriculum Plan Service

## Описание сервиса

Сервис учебного плана предназначен для хранения информации об образовательных программах, дисциплинах и распределении дисциплин по семестрам.

## Функции сервиса

1. Добавление специальности  
2. Добавление дисциплины  
3. Создание учебного плана  
4. Добавление дисциплины в учебный план  

## Сущности

- Specialty — специальность  
- Discipline — дисциплина  
- CurriculumPlan — учебный план  
- CurriculumPlanDiscipline — связь дисциплин и учебного плана  

## Связи

- Specialty → CurriculumPlan: один ко многим  
- CurriculumPlan ↔ Discipline: многие ко многим через таблицу CurriculumPlanDiscipline  

## Нормализация

База данных приведена к третьей нормальной форме:

- все поля атомарны  
- отсутствует дублирование данных  
- нет транзитивных зависимостей  
- связь многие-ко-многим вынесена в отдельную таблицу
